# 开发

### 安装依赖
```shell
yarn
```

### 启动项目
```shell
yarn dev
```

### 新建组件
```shell
yarn new
```

### 构建
```shell
yarn build
```

*构建项目时绕过所有检查👇*
```shell
yarn build -n
```

### 发布
```shell
yarn release
```

*发布项目时忽略版本迭代👇*
```shell
yarn release -i
```

*发布项目时指定迭代的版本为 0.3.25 👇*
```shell
yarn release -m 0.3.25
```

*发布项目时绕过所有检查👇*
```shell
yarn release -n
```

**更多配置项请在 [omni.config.js](https://github.com/omni-door/cli/blob/master/docs/OMNI.zh-CN.md) 中编辑**
